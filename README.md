# AI Prompt App

A Next.js app that sends prompts to Google Gemini AI and displays responses with chat history.

## Setup

1. Clone the repo and run `npm install`
2. Get a free API key from [Google AI Studio](https://aistudio.google.com)
3. Create `.env.local` and add: `GEMINI_API_KEY=your_key_here`
4. Run `npm run dev`

## Features

- Prompt input + submit button
- AI response display
- Loading + error states
- Chat history (saved in browser)
- Clear history button
